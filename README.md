# Ionic 2 Jade Scaffold

#### Installation

> npm install -g cordova ionic

> npm install

> gulp install

> ionic state restore

#### Development

> ionic serve

### Build
> ionic platform add <platform>

> ionic build

#### Configure

To change application's name, change "example" in the following files:
* package.json
* ionic.project
* config.xml
* bower.json
* jade/index.jade
* www/js/app.js
* www/js/routes.js
